datetime库（常考~必考）

1.datetime.datetime

  • 构造一个日期时间对象
  • from datetime import datetime
  •  dt = datetime(2024, 12, 10, 12, 30, 0)

  •  print(dt)

2.datetime.timedelta

  • 表示两个日期时间的时间差
  • delta-间隔
  • td1 = timedelta(days = 1)
  • td2 = timedelta(hours = 2, minutes = 30)

3.dt.strftime

  • 格式化日期格式
  • dt是一个日期时间的对象
  • dt = datetime.now()
  • formatted_date = dt.strftime(‘%Y-%m-%d %H:%M:%S’)
  • %Y表示4位年份，其余都为2位，%m表示月份（01 - 12），%d表示日（01 - 31），%H表示 24 小时制- 的小时（00 - 23），%M表示分钟（00 - 59），%S表示秒（00 - 59）

4.dt.total_seconds

  • timedelta类里的方法
  • 返回时间间隔对象所表示的总秒数
  • from datetime import timedelta
  • td = timedelta(days = 1, hours = 2, minutes = 30)
  • total_secs = td.total_seconds()
  • print(total_secs)

3.counter

  • 在 Python 的collections模块中的Counter类
  • 含义：Counter是一个用于计数的容器类。它继承自dict类，主要用于统计可迭代对象中- 元素出现的次数。例如，对于一个包含多个单词的列表，可以使用Counter快速统计每个- 单词出现的频率。
  • 创建方式：
  • 可以直接从一个可迭代对象（如列表、元组、字符串等）创建Counter对象。
  • python
  • from collections import Counter

  • my_list = ['apple', 'banana', 'apple', 'cherry', 'banana', 'apple']

  • counter_obj = Counter(my_list)

  • print(counter_obj)
  • 输出为Counter({‘apple’: 3, ‘banana’: 2, ‘cherry’: 1})，可以看到它自动统计- 列表中每个元素出现的次数。
